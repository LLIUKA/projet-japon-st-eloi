# 🇯🇵 Japan Tourisme - Site d'Immersion Culturelle

Bienvenue sur le projet **Japan tourisme**, une plateforme web immersive dédiée à la découverte de la culture, de la gastronomie et des paysages du Japon.

L'objectif de ce projet est de proposer une **expérience visuelle et culturelle immersive**, permettant aux utilisateurs d'explorer différents aspects du Japon à travers un Tourisme moderne, une navigation fluide et un contenu riche.

---

# 🚀 À propos du projet

**Japan Tourisme** est un site web statique conçu pour mettre en valeur le patrimoine japonais et faciliter la découverte du pays.

Le projet met l'accent sur :

- une **expérience utilisateur intuitive**
- une **esthétique inspirée du Tourisme japonais**
- une **navigation simple et accessible**
- une **adaptation complète aux différents écrans**

Le site propose plusieurs sections permettant de découvrir :

- la **culture traditionnelle japonaise**
- la **gastronomie locale**
- les **paysages emblématiques**
- des **conseils pour les voyageurs**

---

# ✨ Fonctionnalités principales

### 🌐 Navigation bilingue
Le site propose une interface **Français / Anglais** avec un système de traduction dynamique permettant aux visiteurs de changer de langue instantanément.

### 🖼 Galerie multimédia immersive
Une galerie permettant de découvrir :

- des paysages japonais
- des éléments culturels
- des vidéos immersives

### 📱 Tourisme responsive
Le site est entièrement optimisé pour :

- ordinateurs (desktop)
- tablettes
- smartphones

Grâce à l'utilisation de **Flexbox**, **CSS Grid** et **Media Queries**.

### 🎞 Animations et interactions
Des animations légères ont été intégrées afin d'améliorer l'expérience utilisateur :

- transitions visuelles
- effets de survol
- carrousel sur la page d'accueil

### 📩 Formulaire de contact
Une page contact permettant aux visiteurs de :

- envoyer un message
- poser des questions
- demander des informations supplémentaires

---

# 🛠 Technologies utilisées

Le projet utilise des **technologies web standards**, garantissant compatibilité, performance et simplicité de maintenance.

### HTML5
- structure sémantique du site
- amélioration de l'accessibilité
- meilleure indexation SEO

### CSS3
Utilisé pour :

- la mise en page
- le Tourisme responsive
- les animations

Techniques utilisées :

- **Flexbox**
- **CSS Grid**
- **Media Queries**
- **Transitions CSS**

### JavaScript (Vanilla JS)

JavaScript est utilisé pour :

- la gestion du **changement de langue (i18n)**
- les **interactions utilisateur**
- les **animations dynamiques**
- la logique du **carrousel**
