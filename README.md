# Japan Design - Site d'Immersion Culturelle

Bienvenue sur le projet **Japan Design**, une plateforme web immersive dédiée à la découverte de la culture, de la gastronomie et des paysages du Japon.

## 🚀 À propos du projet
Ce site a été conçu pour offrir une expérience utilisateur fluide et esthétique, mettant en avant le patrimoine japonais à travers :
* Une navigation intuitive et bilingue (Français/Anglais).
* Une galerie multimédia interactive.
* Des sections de conseils de voyage et de culture.
* Un design responsive adapté aux supports mobiles et desktop.



## 🛠 Technologies utilisées
Le projet utilise les technologies web standards pour garantir performance et compatibilité :
* **HTML5** : Structure sémantique du contenu.
* **CSS3** : Mise en forme, Flexbox, Grilles CSS, animations et design responsive via Media Queries.
* **JavaScript (Vanilla)** : Gestion de l'interactivité, basculement de langue (i18n) et logique des composants.
